# Bookstore Project

## Description
Ce projet consiste à concevoir un site web simulant une plateforme de bibliothèque.  
Il permet aux utilisateurs de naviguer à travers des livres, de voir les livres les plus populaires et d’ajouter des livres à un panier virtuel.

Ce projet permet de se familiariser avec l'utilisation des frameworks CSS les plus courants et de créer un site web fonctionnel et responsive.


## Technologies utilisées
- HTML5  
- CSS Framework CSS (Bootstrap)  
- Font Awesome pour les icônes
